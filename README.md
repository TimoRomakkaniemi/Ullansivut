# Ullansivut

Next.js pohjainen verkkosovellus.

## Käyttöönotto

Asenna ensin riippuvuudet:

```bash
npm install
```

Käynnistä kehityspalvelin:

```bash
npm run dev
```

Avaa [http://localhost:3000](http://localhost:3000) selaimessasi nähdäksesi sovelluksen.

## Teknologiat

- [Next.js](https://nextjs.org/) - React framework
- [React](https://reactjs.org/) - JavaScript kirjasto käyttöliittymien luomiseen
- [TypeScript](https://www.typescriptlang.org/) - JavaScriptin laajennos tyypityksellä
